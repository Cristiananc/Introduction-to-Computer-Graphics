Introdução

Padrões estão presentes em todos os lugares, seja nas estampas de roupas que usamos, cerâmicas, ladrilhos, e muitos outros. Padrões são úteis em gráficos quando usados com texturas. Além disso, podem ser usados para criar superfícies diversas como madeira, cortiça, couro e efeitos de ilusão de ótima com aplicações desde ornamentação à estruturas 3D complexas.

Trabalhando com padrões

  1. O primeiro passo é criar texturas;
    2. É possível criar textutas de maneira algoritmica. Uma das técnicas mais simples é chamada tiling (ladrilhamento), onde são colocadas múltiplas cópias de um único formato de acordo com algumas regras, como por exemplo um conjunto de operações de simetria. Uma curiosidade interessante é que apenas 3 polígonos regulares cobrem o plano sem lacunas (gaps) ou sobreposição (overlaps), são eles : triângulo, quadrado e hexágono regular. Há uma quantidade infinita de formas que podemos usar, mas essas são as mais simples.

    Um exemplo utilizando operações simétricas é a criação de um quadrado com a letra F formado por quadradinhos. Há oito modos de colocar esse "ladrilho" paralelo ao plano, contando flip e rotações.

  2. O segundo passo é criar um novo quadrado;
    3. Aqui podemos criar um padrão quadrado com 4 quadrados do passo anterior de 8^4 maneiras. Podemos usar operações com simetria: translação, rotação, reflexão e glide para criar padrões visualmente interessantes, assim podemos criar funções com uma sequência de operações que criam esses padrões simétricos.

Padrões períodicos e aperíodicos

Cada polígono é chamado de "tile", e a composição do padrão é chamado de "tiling". Podemos controlar nossos padrões tiling adotando regras que descrevam de que modos cada tile pode ser colocado ao lado do outro.

No caso dos padrões períodicos há uma célula fundamental que se repete atráves de translações para cobrir o plano. Em geral, em qualquer tiling períodico existe um número infinito de células fundamentais. Há também os padrões quase períodicos, onde pode-se achar grandes repetições de padrões, apesar do padrão completo nunca se repetir.

Em 1961, Alao Wang conjecturou que não haviam tiles aperíodicos.

Pode-se criar também um padrão de encaixe nas arestas ou dispensar esse padrão e utilizar uma decoração representando essas arestas encaixantes.
